## 兵庫県オープンデータ点群データの公開について（水野）

- 兵庫県[オープンデータカタログサイト](http://open-data.pref.hyogo.lg.jp/)
    - [兵庫県_全域数値地形図（2010年度～2018年度）](http://open-data.pref.hyogo.lg.jp/index.php?action=pages_view_main&active_action=multidatabase_view_main_detail&content_id=12517&multidatabase_id=206&block_id=80#_80)が1月10日についに公開されました
    - →活用事例
        - Code for Sapporo　古川さん　[1m精度の高さオープンデータ公開を記念して三ノ宮の街をQGISで3D化してみるよ](https://qiita.com/Yfuruchin/items/2a9b017d840c46c37f3e?fbclid=IwAR3k0NFUqoGvLdARoqlPlvTYUV0EZjJvZkUpXXuE4pcosugzj94aq473xV8)
        - 県OD担当者 近藤様のコメント
    　　「[Cloud Compare](https://www.cloudcompare.org/)」というフリーソフトにドラッグするだけで可視化を超簡単にできました（僕も今朝教えてもらったばかりで、すぐにできました）。グーグルアースみたいにぐりぐり動かせて面白いです。地形マニアや古墳マニア、棚田マニアの方など、ぜひ触ってみてください！
- Code for Harimaに県の近藤様が説明に来ていただけるとか
    - →依頼する
