## Code for HarimaのWebサイトを作っています（のがた）

- 正月休みでする予定だった仕事ができなくなったので、空いた時間でCode for
  HarimaのWebサイトを作っていました
  - リポジトリ: <https://github.com/codeforharima/website>
  - 作った理由は時間が空いたこともありますが、発表の資料を作るために昔の活動を調べてたら、活動の履歴をちゃんと見せたほうがいいなと感じたこともあります
  - システムは、[Pelican](https://getpelican.com/)という[静的サイトジェネレーター](https://www.cloudflare.com/ja-jp/learning/performance/static-site-generator/)を使っています
    - 静的サイトジェネレーターというのは、このHackMDのMarkdownファイルをアップロードするとHTMLに変換してWebサイトとして見られるようになるシステムです
  - 今はHTMLのモックからPelicanのテンプレートに起こしている最中
    - モックはAIを使って作ったけど、デザイン関連は微妙で指示をたくさん書くハメになったので途中から自分で手直ししてました…
  - 予定としては、テンプレートが完成したら、Markdownから変換するためのGithub
    Actionsの設定をすれば見られるようになるはず
    - 完成すれば、議事録のMarkdownファイルをアップロードするだけで毎月の記録ができて、ブログも書けるようにしてあるのでFacebookページ以外でもお知らせできるはず
  - サイトのURLは、codeforharima.github.ioになる予定ですが、[独自ドメインの設定もできる](https://docs.github.com/ja/pages/configuring-a-custom-domain-for-your-github-pages-site)ので、どうしましょ
